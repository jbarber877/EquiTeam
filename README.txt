### Project Overview
This website is our submission for the GEA final group project in EGN3615. It focuses on **UN Sustainable Development Goal #7: Affordable and Clean Energy**. 
The site includes a homepage and several interactive tools that address the key aspects of affordable and clean energy.

### Usage Instructions
1. **Navigation**: Each tool can be accessed by clicking the “Learn More” button in its respective section on the homepage.
2. **Running the Tools**: After navigating to a tool, click the **Run** button (triangle icon) to activate it.
